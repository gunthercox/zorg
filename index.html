import time
import zorg

def move_servo(my):
    angle = 0

    while True:
        my.servo.set_angle(angle)

        angle += 45

        if angle > 135:
            angle = 0

        time.sleep(0.1)

robot = zorg.robot({
    "name": "Test",
    "connections": {
        "edison": {
            "adaptor": "zorg_edison.Edison",
        },
    },
    "devices": {
        "led": {
            "connection": "edison",
            "driver": "zorg_gpio.Servo",
            "pin": 5, # Digital/PWM pin 5
        },
    },
    "work": move_servo,
})

robot.start()

Configure your robots with Zorg!

Zorg is a Python framework for robotics, and Internet of Things projects. It makes it easy to configure and command your projects.

Zorg runs on a variety of platforms, including the Intel Edison and the Raspberry Pi. There is support for a variety of analog and digital sensors built in, and Zorg is extensible so that anyone can add support for new platforms and devices.
